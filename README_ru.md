# Предсказание сердечно-сосудистых заболеваний (ML + Flask)

Данный проект является дипломной работой, выполненной в рамках обучения в **Proweb Data Science**.

Проект посвящён прогнозированию риска сердечно-сосудистых заболеваний с использованием моделей машинного обучения и веб-приложения.

---

## 📊 Обзор проекта

Цель проекта — создать модель машинного обучения, которая определяет риск наличия сердечно-сосудистого заболевания на основе медицинских и поведенческих факторов.

Проект включает:
- Анализ данных
- Обработку данных
- Обучение и сравнение моделей
- Развертывание модели через Flask-приложение

---

## 🗂 Датасет

- Источник: публичный (Kaggle / open data)
- Количество записей: ~70 000
- Признаки: 12 входных + 1 целевой

### Целевая переменная:
- `cardio`
  - 0 — нет заболевания
  - 1 — есть заболевание

### Признаки:
- Возраст (в днях)
- Пол
- Рост / вес
- Артериальное давление
- Холестерин
- Глюкоза
- Курение
- Алкоголь
- Физическая активность

---

## ⚙️ Анализ и обработка данных

- Очистка данных
- Анализ признаков
- Расчет и анализ BMI
- Проверка баланса классов (~50/50)
- Отсутствие пропущенных значений

---

## 🤖 Модели машинного обучения

Были протестированы:
- Random Forest
- KNN (k=5, k=9)
- Logistic Regression ✅ (выбрана)

### Почему Logistic Regression?

- Подходит для бинарной классификации
- Хорошо работает с медицинскими данными
- Менее склонна к переобучению
- Быстро обучается
- Выдает вероятности (важно для оценки риска)

---

## 📈 Задача модели

Определить вероятность наличия заболевания и классифицировать пациента:

- Здоров
- Есть риск

---

## 🌐 Веб-приложение (Flask)

Разработано веб-приложение на Flask для удобного использования модели.

### Возможности:
- Форма ввода данных пациента
- Мгновенное предсказание
- Удобный интерфейс
- Поддержка 3 языков:
  - Русский
  - Английский
  - Узбекский

### Как работает:
1. Пользователь вводит данные
2. Данные передаются в модель
3. Модель делает прогноз
4. Результат отображается на экране
